## beamer

导入ctexbeamer自动就有中文环境

```latex
\documentclass[UTF8]{ctexbeamer}
\documentclass[UTF8,draft]{ctexbeamer} # 不编译图片和超链接，加快预览编译
```

beamer样式选择见https://mpetroff.net/files/beamer-theme-matrix/

具体可以看得更清晰：

主题：https://blog.csdn.net/zhouxiaowei1120/article/details/82818295

```
\usetheme{Frankfurt}
```

颜色：https://blog.csdn.net/zhouxiaowei1120/article/details/82818348

```
\usecolortheme{beaver}
```
